出力タブ

これは、一連のサブタブでさまざまな情報メッセージを表示します。

「全般」タブが1つあり、予期しない例外のスタックトレースが含まれる場合があります - バグを報告する際は、スタックトレースを含めてください。

その他のZAPアドオンは、独自のサブタブを追加できます。特に「スクリプトコンソール」アドオンは、出力を生成する各スクリプトに対して1つのタブを追加します。
