アラートタブ

アラートタブには、このセッションで生成されたアラートが表示されます。 アラートは左ペインにリスク順のツリー形式で表示され、ツリーの各ノードにはその下に存在するアラートの合計数が示されます。

アラートを一度クリックして選択すると、右ペインにその詳細が表示されます。 アラートをダブルクリックすると、アラートの追加ダイアログが表示され、アラートの詳細を変更できます。

アラートのフィルタリング

ツリーの直上に表示されているフィルターボタンを使用すると、表示されるアラートを制限できます：

 /    スコープ内のURLのみ表示 / すべてのURLを表示

スコープ内のURLのアラートのみを表示できます.

 /    サイト選択とリンク / リンク解除

サイトタブ で選択されたツリーノードのアラートのみを表示できます。

注意: 一方のフィルターを選択すると、他方のフィルターは相互排他的であるため無効になります。

  アラートの編集

選択されたアラートの詳細を変更できる アラートの追加 ダイアログが表示されます。

注意: ツリー内で複数のアラートが選択されている場合、最も新しく選択された項目が編集されます。

  すべてのアラートを削除

すべてのアラートを削除できます。
