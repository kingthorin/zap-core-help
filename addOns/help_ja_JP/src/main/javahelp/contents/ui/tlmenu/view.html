表示メニュー

このメニューは表示オプションを処理します。

画像イメージも履歴に記録

If selected then images will be displayed in the 履歴タブ.

Show Tab

非表示にしているものを含む、どんな利用可能なタブも表示する事ができます。

Show All Tabs

This allows you to show all tabs.

Hide Unpinned Tabs

This allows you to hide all tabs that are not pinned.

Pin All Visible Tabs

This allows you to pin all visible tabs.

Unpin All Tabs

This allows you unpin all tabs.

Note that add-ons can add additional menu items.
