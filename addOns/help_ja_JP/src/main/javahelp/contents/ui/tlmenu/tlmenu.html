トップレベルメニュー

By default the following top level menu items will be present:
     ファイルメニュー This handles the current session
     Edit menu This handles finding strings in specific tabs, searching for strings across all requests and responses, changing ZAP's mode, and managing forced user mode
     View menu This handles the display options
     Analyse menu This handles Scan Policies
     レポートメニュー This handles the reports
     Tools menu This handles the tools and general options
     Import menu This gives access to menus that allow to import data into ZAP.
     Online menu This gives access to a set of online resources
     Help menu This gives access to the 'about' dialog and this help file

Note that アドオン can add additional menu items.
