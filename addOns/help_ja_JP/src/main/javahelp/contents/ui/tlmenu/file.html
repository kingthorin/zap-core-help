ファイルメニュー

このメニューは現在のセッションを処理します。デフォルトでは以下のメニュー項目が表示されます：

新規セッション

新しいセッションを作成します。
現在のセッションを保存していない場合、警告が表示されます。
現在のセッションを保存せずに新しいセッションを開始すると、現在のセッションのすべてのデータが失われます。

セッションを開く...

以前に保存されたセッションを開きます。
現在のセッションを保存せずにセッションを開くと、現在のセッションのすべてのデータが失われます。

最近使用したファイルを開く

最近アクセスしたファイルの一覧です。

セッションを保存…

現在のセッションを永続化します。
セッションは常にディスク上のデータベースに保存されますが、永続化されていない場合、ZAP を終了すると失われます。
永続化は一度だけ行えば十分です。それ以降の変更はすべて保存されます。

セッションのスナップショットを作成…

既に永続化されているセッションのスナップショットを保存します。
永続化されているセッションと同じファイル名に日時文字列を追加したファイル名が提案され、ユーザーが任意の名前を設定できます。

セッションプロパティ…

セッションプロパティ ダイアログを表示します。
セッションの名前と説明を設定します。

コンテキストをインポート...

コンテキストをインポートできます.

コンテキストをエクスポート...

コンテキストをエクスポートできます。

アドオンファイルをロード…

ローカルの アドオン ファイルを読み込みます。
アドオンは通常 アドオンの管理 ダイアログからインストールされますが、このオプションはアドオンを手動でダウンロードした場合や、自分で開発したアドオンをテストする場合に便利です。
アドオンは手動でアンインストールするまでインストールされたままになります。

セッションを削除して終了…

ZAPを終了し、以前に保持していたとしてもセッションを削除します。
ZAPを再起動してもセッションにアクセスできなくなりますが、作成したスナップショットは引き続き利用できます。
このオプションを本当に選択するつもりだったかを確認する警告ダイアログが表示されます。

終了

ZAPを終了します。
現在のセッションを保存していない場合、保存するオプションが提供されます。

アドオンにより追加のメニュー項目が追加される場合があります。
