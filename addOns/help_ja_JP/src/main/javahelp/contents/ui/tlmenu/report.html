レポートメニュー

このメニューはレポートを処理します。

他セッションとの比較...

以前に保存したZAPセッションを選択するよう求められます。
次に、結果を書き込むための出力ファイルを指定します。このファイルには、現在のセッションと比較対象セッションの両方でアクセスされた全てのURLが記録されます。
出力ファイルには、2つのセッションにおけるURLと、各URLに対するHTTPレスポンスの一覧表が含まれます。
JavaScriptボタンを使用して表示を絞り込むことができます: 全てのURL、最初のセッションのみ、2番目のセッションのみ、両方のセッションに共通のURL。
この機能は、異なる権限を持つユーザーで同じアプリケーションにアクセスした2つのセッションを比較する際に特に役立ちます。 これにより、各ユーザーに表示されるURLを確認できるほか、どちらか一方のユーザーとしてログインした状態で、全てのURLへのアクセスを試みることが可能になります。

アドオンにより追加のメニュー項目が追加される場合があります。
