編集メニュー

このメニューは、特定のタブ内での文字列検索、すべてのリクエストとレスポンスにわたる文字列の検索、ZAPのモード変更、および強制ユーザーモードの管理を処理します。

ZAP モード

現在の モードを変更できます.

検索…

これにより 検索ダイアログ が開き、現在選択されているウィンドウ内で文字列を検索できます。

検索…

このサブメニューでは以下の項目にアクセスできます：

検索…

これにより 検索タブ が選択され、すべてのURL、リクエスト、レスポンスで正規表現を検索できます。

次の結果

最後に検索した文字列の次の出現箇所を選択します。
関連するメッセージが検索タブで選択され、文字列が適切な リクエスト または レスポンス タブに表示・ハイライトされます。

前の結果

最後に検索した文字列の前の出現箇所を選択します。
関連するメッセージが検索タブで選択され、文字列が適切な リクエスト または レスポンス タブに表示・ハイライトされます。

強制ユーザーモードの有効化 / 無効化

これにより 強制ユーザーモード のオン・オフを切り替えます。
このメニュー項目は、少なくとも1つの コンテキストに対して強制ユーザーを定義した場合のみ有効になります。これは セッションコンテキスト ダイアログから実行できます。

アドオンにより追加のメニュー項目が追加される場合があります。
