オプション - アップデート確認画面

この画面では、ZAP 起動時に最新バージョンかどうかをチェックするか、その際に何を行うかを設定できます。

起動時に更新を確認する

このオプションを選択すると、ZAPは起動時にアプリケーションとすべてのアドオンのアップデートを自動的に確認します。
この設定を有効にすることが強く推奨されています。
何らかの理由でこの設定を無効にする場合は、手動で頻繁にアップデートを確認する必要があります。
ZAPはアップデート確認のために1つのリクエストのみを行い、そのリクエストに含まれる情報は現在のバージョン番号のみです。

新しいZAPリリースを自動ダウンロードする

このオプションを選択すると、ZAPは新しいリリースが利用可能になった際に自動的にダウンロードし、インストールを促します。

インストール済みアドオンを自動アップデートする

このオプションを有効にすると、ZAPはインストール済みのアドオンのアップデートを自動的にダウンロードし、インストールします。

インストール済みスキャンルールを自動アップデートする

このオプションを選択すると、ZAPはインストール済みスキャンルールのアップデートを自動的にダウンロードし、インストールします。
これらは、インストールされているアドオン全体のサブセットです。

新しいリリース版アドオンを通知する

このオプションを選択すると、アドオンが「リリース」ステータスに昇格した場合にZAPが通知します。
これらのアドオンは徹底的にテストおよびレビューされており、最高の品質が保証されています。

新しいベータ版アドオンを通知する

このオプションを選択すると、アドオンが「ベータ」ステータスに昇格した場合にZAPが通知します。
これらのアドオンはテストとレビューが行われており、合理的な品質で、ほとんどの目的に適していると考えられています。
しかし、未完成であったり、さらなるテストが必要な場合があります。
ZAPにデフォルトで含まれている一部のアドオンも、まだベータ版レベルです。

新しいアルファ版アドオンを通知する

このオプションを選択すると、アドオンが「アルファ」ステータスで追加された場合にZAPが通知します。
これらのアドオンはレビューされていますが、通常は開発の初期段階にあります。
未完成であったり、重大な問題を含んでいたり、安定性の問題を引き起こす可能性があります。

アドオンディレクトリ

ZAP will load all of the add-ons from the デフォルト plugin ディレクトリ.
必要に応じて、ここに追加のアドオンディレクトリを追加できます。
これは、CI環境などで1台のマシンで複数のZAPインスタンスを実行する場合に非常に便利です。
これにより、新しいアドオンや更新されたアドオンを一度ダウンロードするだけで、複数のインスタンス間で即座に共有できます。
共有ディレクトリにダウンロードされたアドオンは、ダウンロードを実行したインスタンスを除き、既に実行中のZAPインスタンスには追加されず、新しいインスタンスにのみ追加されます。

ダウンロード ディレクトリ

これは、ZAPが新しいアドオンや更新されたアドオンをダウンロードするディレクトリです。
デフォルトではローカルディレクトリですが、このZAPインスタンスが書き込み可能であれば、共有ディレクトリのいずれかを選択することもできます。
ZAPは、アドオンが完全にダウンロードされてから共有ディレクトリに移動させるため、他のZAPインスタンスが不完全なアドオンを読み込もうとすることはありません。
