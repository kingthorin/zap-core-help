セッションコンテキスト構造画面

これは セッションコンテキスト画面 の一つで、アプリケーションの構造に対するZAPの解釈を管理できます。

URL キー/値ペアの区切り文字

URLにおいてキーと値のペアを区切る文字です。デフォルトは「&」です。

URL キーと値の区切り文字

URLにおいてキーと値を区切る文字です。デフォルトは「=」です。

POST キー/値ペアの区切り文字

POSTデータにおいてキーと値のペアを区切る文字です。デフォルトは「&」です。

POST キーと値の区切り文字

POSTデータにおいてキーと値を区切る文字です。デフォルトは「=」です。

構造修飾子

このコンテキストに適用される 構造修飾子 には以下があります：
  • データ駆動型コンテンツ - データを表すURLパスを識別します
  • 構造パラメーター - ユーザーデータではなくアプリケーション構造を表すパラメータを識別します
