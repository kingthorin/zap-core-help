オプション データベース画面

この画面では、データベースのオプションを設定することができます。

終了時にコンパクト化

ZAP終了時にデータベースを圧縮します。圧縮により ディスク使用量を最小限に抑えられますが、ZAPの終了時間が長くなります。

リカバリーログ

データベースのリカバリーログを有効にするかどうかを制御します。
無効にするとデータベースのパフォーマンスは向上しますが、ZAPが予期せず終了した場合に未保存のデータが失われる可能性があります。
注意: 現在のセッションには影響せず、新規または再読み込みされたセッションに適用されます。

リクエストボディの最大サイズ

ZAPが許容するリクエストボディの最大サイズ (バイト)。

レスポンスボディの最大サイズ

ZAPが許容するレスポンスボディの最大サイズ (バイト)。

新規セッションの保存オプション

有効にすると、新しいセッション作成時に セッション永続化ダイアログ が表示されます。

デフォルトオプション

新しいセッションのデフォルトオプション。上記のプロンプトが有効な場合、このオプションが初期選択されます。
プロンプトが無効な場合は、このアクションが自動実行されます。
