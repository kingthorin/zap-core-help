オプション - スクリプト画面

この画面では、スクリプトオプションを設定できます：

ディレクトリから読み込まれたスクリプトを有効化

このオプションを選択すると、指定されたディレクトリから読み込まれた全てのスクリプトがデフォルトで有効になります。

ディレクトリ

スクリプトが読み込まれるディレクトリのリストです。ZAPは文字エンコーディングUTF-8を使用してスクリプトを読み込み (および書き込み) ます。

スクリプトは、関連するスクリプトタイプ ('active'、'passive'、'proxy'など) に対応した名前のサブディレクトリに配置する必要があり、使用するスクリプト言語に応じて適切な拡張子を持つ必要があります。

読み取り専用のスクリプトは、「テンプレート」セクションに追加されます。

ZAPコミュニティスクリプトリポジトリを追加することをお勧めします: https://github.com/zaproxy/community-scripts

ローカルコピーをクローンする必要がありますが、その後はコミュニティが追加した全てのスクリプトにアクセスでき、GitHubプルリクエストを通じて独自のスクリプトをアップロードすることもできます。
