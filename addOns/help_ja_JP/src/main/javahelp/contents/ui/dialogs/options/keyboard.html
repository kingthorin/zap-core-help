オプション キーボード画面

この画面では、ZAPのすべてのメニューに対するキーボードショートカットを設定できます。

テーブルのいずれかの行をクリックすると、キーボードショートカットを変更するためのダイアログボックスが表示されます。

「アクション」チートシート

このボタンをクリックすると、「アクション」順に並べ替えたすべてのキーボードショートカットを一覧表示するHTMLページが生成されます。
このボタンは、APIが有効で、かつお使いのOSがブラウザでURLを開くことをサポートしている場合にのみ有効になります。

「キー」チートシート

このボタンをクリックすると、「キー」順に並べ替えたすべてのキーボードショートカットを一覧表示するHTMLページが生成されます。
このボタンは、APIが有効で、かつお使いのOSがブラウザでURLを開くことをサポートしている場合にのみ有効になります。

リセット

このボタンをクリックすると、すべてのショートカットがデフォルト値にリセットされます
