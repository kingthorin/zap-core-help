履歴フィルターダイアログ

このダイアログでは、履歴タブに表示されるリクエストを制限できます.

フィールド

このダイアログには以下のフィールドがあります：

メソッド

フィルター対象のHTTPメソッド

コード

フィルター対象のHTTPレスポンスコード

タグ

フィルター対象の タグ

アラート

フィルター対象の アラート タイプ

ノート

ノート の有無に基づいてフィルターをかけるかどうか

URL (含める正規表現)

「含める」正規表現のリスト (1行に1つ)。指定された場合、そのパターンに一致するURLのみが含まれます。

URL (除外正規表現)

「除外する」正規表現のリスト (1行に1つ)。指定された場合、そのパターンに一致するURLはすべて非表示になります。
