オプション - 統計画面

この画面では、ZAPが保持する統計情報を設定できます。

メモリ内統計情報を有効にする

デフォルトでは、ZAPは統計情報をメモリ内に保持します。メモリ内統計を無効にするには、このボックスのチェックを外してください。

Statsdサーバーを有効にする

ZAPが統計情報を送信するStatsdサーバーのアドレスを設定するには、このボックスにチェックを入れてください。

Statsdホスト名

Statsdサーバーが稼働しているホストです。

Statsdポート

Statsdサーバーがリッスンしているポートです。デフォルトは8125です。

Statsdプレフィックス

ZAPがStatsdサーバーに送信する統計情報の先頭に付けるプレフィックスです。
