オプション - ルール設定画面

この画面では、特定のアクティブスキャンルールおよびパッシブスキャンルールの挙動を設定できます。

リストされたいずれかの行を選択すると、特定の設定の詳細が表示され、関連する値を変更できます。 個々のルール、およびすべてのルールをリセットできます。

組み込みのルールには、以下のものがあります：

名前既定値説明
rules.common.sleep15タイミング攻撃に使用される時間 (秒)
rules.cookie.ignorelistCookie名のカンマ区切りリスト このリストに含まれるCookieは、Cookie関連の問題をスキャンする際に無視されます。
rules.csrf.ignorelist識別子のカンマ区切りリスト これらの識別子のいずれかに一致する name または ID を持つFORMは、Anti-CSRFトークンの欠落をスキャンする際に無視されます。 この機能は、検索フォームなど、安全であることが確認されているFORMを無視する場合にのみ使用してください。
rules.csrf.ignore.attnameフォームにAnti-CSRFトークンが不要であることを示すために使用できるHTML属性の名前 rules.csrf.ignore.attvalue が指定されている場合、属性の値も一致する必要があります。 見つかった場合、関連するアラートは情報 (INFO) レベルで報告されます。
rules.csrf.ignore.attvaluerules.csrf.ignore.attname で指定されたHTML属性の値で、フォームにAnti-CSRFトークンが不要であることを示すために使用できます。 見つかった場合、関連するアラートは情報 (INFO) レベルで報告されます。
rules.domains.trustedURLの正規表現パターンのカンマ区切りリスト。このパターンに一致するURLは信頼済みドメインと見なされ、関連する問題は無視されます。
