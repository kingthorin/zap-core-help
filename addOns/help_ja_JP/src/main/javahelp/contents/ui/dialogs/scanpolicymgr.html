スキャンポリシー管理ダイアログ

This allows you to manage the スキャンポリシー that define the rules that are run when performing an アクティブスキャン.
スキャンポリシーは必要な数だけ作成でき、アクティブスキャンを実行する際にどのポリシーを使用するかを選択できます。

追加

「追加」ボタンを押すと、すべての設定がデフォルト値の状態で スキャンポリシーダイアログ が起動します。

変更

「変更」ボタンを押すと、選択したスキャンポリシーの設定が読み込まれた状態で スキャンポリシーダイアログ が起動します。

削除

「削除」ボタンを押すと、警告ダイアログで確認した後、選択したスキャンポリシーが削除されます。

インポート

「インポート」ボタンを使用すると、ZAPの他のインスタンスで作成されたスキャンポリシーをインポートできます。

エクスポート

「エクスポート」ボタンを使用すると、スキャンポリシーをエクスポートできます。エクスポートしたポリシーは、別のマシン上にあるZAPのインスタンスにインポートすることが可能です。
