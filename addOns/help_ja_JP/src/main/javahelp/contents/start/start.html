はじめに

ZAPを使い始める最も簡単な方法は、デフォルトでインストールされているクイックスタートアドオンを使用することです。
これにより、URLを入力するだけで、ZAPが最初にスパイダーを実行し、次にアクティブスキャンを実行します。
より詳細なテストを行うには、ZAPをプロキシとして経由させながら、ブラウザまたは自動回帰テストを使用してアプリケーションを探索する必要があります。

ZAPの本質はManipulator-in-the-Middleプロキシです。
ZAPを最大限に活用するには、テストしたいWebアプリケーションにZAP経由で接続するようにブラウザまたは機能テストを設定する必要があります。
必要に応じて、別のプロキシ経由で接続するようにZAPを設定することもできます - これは企業環境でよく必要になります。

クイックスタートタブから、ZAP経由でプロキシするように事前設定されたブラウザを迅速かつ簡単に起動できます。 これにより、新しいプロファイルでインストールされている最も一般的なブラウザを起動できます。
他のブラウザアドオンがインストールされている既存のプロファイルなど、既存のプロファイルでブラウザを使用したい場合は、ZAP経由でプロキシするようにブラウザを手動で設定する必要があります。
Webブラウザでプロキシを設定する方法を知っている場合は、ぜひ試してみてください！
不明な場合は、プロキシの設定セクションを参照してください。

ZAPをブラウザのプロキシとして設定したら、テストするWebアプリケーションに接続してみてください。
接続できない場合は、プロキシ設定を再度確認してください。 ブラウザのプロキシ設定とZAPのプロキシ設定の両方を確認する必要があります。
テストしようとしているアプリケーションが実行されているかどうかも確認する価値があります！

ブラウザ経由でアプリケーションに正常に接続できたら、ZAPをもう一度見てください。 サイトタブと履歴タブに1つ以上の行が表示されているはずです。
表示されていれば順調です。 表示されていない場合は、プロキシ設定を再度確認する必要があります。

次にすべきことは、基本的な侵入テストを開始することです。.
