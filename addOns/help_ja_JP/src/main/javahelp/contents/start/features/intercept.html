Manipulator-in-the-middle Proxy

ZAP is a Manipulator-in-the-middle Proxy. あなたがWebアプリに対して行ったリクエスト全て、および受信したレスポンスの全てを見る事が出来ます。
とりわけ、明らかではないかもしれない、AJAXの呼び出しを見る事が出来ます。

You can also set breakpoints which allow you to change the requests and responses on the fly.
