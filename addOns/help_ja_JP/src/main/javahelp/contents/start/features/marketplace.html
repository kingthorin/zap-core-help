マーケットプレイス

ZAP マーケットプレイスには、ZAPチームとコミュニティによって作成されたZAPアドオンが含まれています。 アドオンはZAPの機能を拡張するのに役立ちます。 ツールバーにある 「アドオンの管理」ボタンをクリックし、次にマーケットプレイスタブを選択することで、ZAP内からアドオンを閲覧およびダウンロードできます。

ZAP マーケットプレイスのすべてのアドオンはもちろん無料でオープンソースです。
