構造修飾子

構造修飾子は、ZAPがアプリケーションの構造をどのように表現するかを変更するコントロールです。
サイトツリー は、ZAPによるアプリケーションの表現です。
サイトツリーが構造を適切に表現していない場合、ZAPはアプリケーションを効果的に攻撃できません。
現在、構造修飾子には2つのタイプがあります：

  • データ駆動型コンテンツ - データを示すURLパスを識別します
  • 構造パラメーター - ユーザーデータではなくアプリケーション構造を表すパラメータを識別します
構造修飾子は コンテキストにのみ適用できます.
