アドオン

アドオンによって、ZAPに機能を追加することができます。
アドオンはZAPの内部機能を完全に利用することによって、非常に強力な新機能を提供することが可能です。
アドオンの管理ダイアログから、オンラインのアドオンマーケットプレースにあるアドオンを動的にインストールすることができます。
通常、ZAP UIへのアドオンの追加と削除は再起動なしに行うことができます。

アドオンはヘルプファイルと動的に統合される、コンテキストに応じたヘルプを提供できます。
このページを（オンラインではなく）ZAPのヘルプファイル経由で閲覧している場合、インストールしたアドオンに関連するすべてのヘルプページがこちらで利用できます.

アドオンは以下のステータスのどれかが割り当てられます。
     Release信頼性が高く、目的を満たしていることを示す
     Betaある程度の信頼性ではあるものの、不完全もしくはさらなるテストが必要であることを示す
     Alpha開発の初期段階であることを示す

ZAPで利用可能なアドオンの場所

アドオンをZAPで利用できるようにするには、次のいずれかの場所にある必要があります。
  • ZAPのインストールフォルダー内にある plugin ディレクトリ
  • ZAPのユーザーフォルダー内にある plugin ディレクトリ
