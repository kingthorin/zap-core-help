ソフトウェア部品表 (SBOM)

ZAPには、ZAPコアとZAPチームが保守するすべてのアドオンについて、CycloneDX形式で生成されたランタイムSBOMが含まれています。 各SBOMは、ZAPのJARファイルのルートにbom.jsonというファイルとして格納されています。

ソースコードからZAPを実行する場合、SBOMが利用できない可能性があり、一部のサードパーティ製アドオンも定義していない場合があることに注意してください。

SBOM zip

利用可能なすべてのSBOMファイルを含むzipファイルは、以下のオプションから生成できます。 ZAPコアのSBOMファイルは "zap-core-bom.json" という名前になり、アドオンのSBOMファイルは "<addon-id>bom.json" という名前になります。

デスクトップ

ヘルプメニューの「サポート情報…」ダイアログにある「SBOMをZIP形式で保存…」ボタン。

コマンドライン

"-sbomzip" コマンドラインオプション。

API

coreの "createSbomZip" APIアクション。
