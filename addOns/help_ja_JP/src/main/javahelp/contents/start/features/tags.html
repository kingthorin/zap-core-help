タグ

タグとは、リクエストに関連付けたい短いテキストのことです。
一つのリクエストには、複数のタグを関連付けることができます。
タグの追加や変更は、履歴タグ管理ダイアログを使用して行えます。
また、パッシブスキャンを介して自動的に追加することもできます。これにより、特定の正規表現に一致するページに自動でタグ付けが可能です。

タグは履歴タブに表示され、履歴フィルターダイアログを使用して絞り込むことができます。.

アラートタグ

履歴タグが ALERT-TAG: で始まる場合、そのタグは当該リクエストに対して発生したアラートにも関連付けられます。接頭辞は削除され、残りの文字列に = 文字が含まれる場合、アラートタグのキーは最初の = までの文字列となり、値は残りの文字列となります。 = がない場合、キーは接頭辞以降の文字列全体となり、値は空になります。
例えば、履歴タグが ALERT-TAG:AAA=BBB の場合、キーが AAA で値が BBBのアラートタグが生成されます。一方、履歴タグが ALERT-TAG:CCC の場合は、キーが CCC で値が空のアラートタグが生成されます。
