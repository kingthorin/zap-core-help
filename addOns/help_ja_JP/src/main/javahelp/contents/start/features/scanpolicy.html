スキャンポリシー

スキャンポリシーは、アクティブスキャンの一部として実行されるルール (脆弱性チェック) を正確に定義します。
また、これらのルールの実行方法を定義し、送信されるリクエストの数や潜在的な問題が検出される可能性に影響を与えます。
任意の数のスキャンポリシーを定義でき、アクティブスキャンダイアログからスキャンを開始する際に最も適切なポリシーを選択できます。
アクティブスキャンおよび攻撃モードで使用されるデフォルトのスキャンポリシーは、オプション アクティブスキャン画面で定義できます.

アクティブスキャンはそれらのターゲットへの攻撃です。
所有していないWebアプリケーションに対して使用しないでください。

アクティブスキャンでは特定の種類の脆弱性しか検出できない点に注意が必要です。
アクセスコントロールの不備といった論理的な脆弱性は、いかなるアクティブスキャンや自動化された脆弱性スキャンでも見つかりません。
あらゆる種類の脆弱性を見つけるには、アクティブスキャンに加えて、必ず手動によるペネトレーションテストを実施してください。

アクティブスキャンは、オプション アクティブスキャン画面を使用して設定します。
また、任意の数のスキャンポリシーを定義できます - これらは実行されるルールとその動作方法を正確に定義します。
