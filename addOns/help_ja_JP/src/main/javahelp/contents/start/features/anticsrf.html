Anti-CSRF トークン

Anti-CSRFトークンは、クロスサイトリクエストフォージェリ (CSRF) 攻撃から保護するために使用される (擬似) ランダムパラメーターです。
しかし、このトークンはペネトレーションテスターの仕事を困難にします。特に、フォームがリクエストされるたびにトークンが再生成される場合はなおさらです。

ZAPは、属性名のみによってAnti-CSRFトークンを検出します。Anti-CSRFトークンと見なされる属性名のリストは、オプション Anti-CSRF画面で設定します。
ZAPがこれらのトークンを検出すると、その値とトークンを生成したURLを記録します。
アクティブスキャナーなどの他のツールには、必要に応じてZAPがトークンを自動的に再生成するためのオプションがあります。
