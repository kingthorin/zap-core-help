スコープ

スコープはテスト対象のURL範囲を定義し、指定したコンテキストによって定義されます。

デフォルトでは、何もスコープに含まれていません。

スコープは以下に影響を与える可能性があります：

  • プロテクトモードの場合、実行できること
  • 履歴タブ に表示される内容
  • プロテクト - スコープ内のURLに対してのみ、危険な可能性のあるアクションを実行できます
  • 標準 - 以前のリリースのように、何でも出来ます
  • 攻撃 - スコープ内の新しいノードは、発見され次第アクティブスキャンされます
テストしているシステムを構成する各Webアプリケーションに対して新しい コンテキスト を定義し、それぞれをテストする際にスコープに含めることを推奨します。
