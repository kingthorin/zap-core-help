モード

ZAPには「モード」があり、以下のいずれかを設定できます：

  • セーフ - 潜在的に危険な操作は許可されません。
  • プロテクト - スコープ内のURLに対してのみ、（潜在的に）危険なアクションを実行できます.
  • 標準 - 何も制限しません
  • 攻撃 - スコープ内の新しいノードが発見されるとすぐにアクティブスキャンが実行されます。
意図したサイトのみを攻撃対象とすることを確実にするため、プロテクトモードを使用することが推奨されます。

モードは ツールバー (またはZAP API) を介して変更でき、セッション間でその設定は維持されます。

セーフモードの場合、またはプロテクトモードでスコープ外のURLを操作している場合に不可能となる操作の例：
  • スパイダー
  • アクティブスキャン
  • ファジング
  • 強制ブラウジング
  • ブレーク (インターセプト)
  • リクエストの再送信
攻撃モードで使用する スキャンポリシー は、 オプション アクティブスキャン画面で定義できます.
