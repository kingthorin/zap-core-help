リリース 1.0.0

本リリースでは以下のような変更が行われました。

重要な変更

ヘルプ画面

ZAPが提供する全ての機能について、ヘルプ画面が追加されました。
F1 キーを押すと選択しているタブに関連するヘルプ画面が表示されるよう、状況依存型ヘルプも実装されました。

ブレークポイント

次の修正を行いました。 ブレークポイント はソフトウェア統合開発環境で提供されるようなものに近づいた形に変更されました。
The breakpoint controls have been moved to the new 「トップレベルツールバー」.
Breakpoints can now also be set by right clicking nodes in the サイト 及び 履歴タブ.

レポートメニュー

　 レポートメニュー は大幅に変更され、次のメニューアイテムを含むようになりました。
    レポートを生成する.今すぐレポートを生成する。
    メッセージをファイルにエクスポート...[ファイル] メニュー から移動しました。
    レスポンスをファイルにエクスポート...[ファイル] メニュー から移動しました。
    全てのURLをファイルにエクスポート...新機能
    他セッションとの比較...新機能

履歴検索の改善

新しい 検索タブ では、すべてのURL、リクエスト、およびレスポンスに対して正規表現で検索できます。
「次」ボタンと「前」ボタンで、見つかった検索文字列を移動できます。 リクエスト 及び レスポンスタブ.

履歴フィルター

履歴フィルターツールバー が新しく追加され、表示するリクエストをフィルタできます。
リクエストは次の単位でフィルタリングできます。HTTPメソッド、HTTPステータスコード、 タグ, アラートまたは ノート （適用されているもの）。

ノート

今では ノート を任意のリクエストに関連付けることができます

マルチタグ

今では複数の タグ を任意のリクエストに関連付けることができます。

アラート全体に対する制御の追加

　 アラートの追加ダイアログ アラートを手動で追加または変更することができます。

新しいエンコーダー/デコーダー

A new Encode/Decode/Hash dialog has replaced the previous Encoder/Decoder.

パッシブスキャン

パッシブスキャン has been added.
In this release ZAP will only use passive scanning for automatically adding タグ.
In future releases it may also raise アラート for potential issues.

小変更：

トップレベルツールバー

The Top Level Toolbar has been added which provides a set up buttons for commonly used functionality.

アラートフッターの概要

Counts of the High, Medium, Low and Informational アラート are now displayed in the footer.

新しい見た目と操作感

A new look and feel 'Nimbus' is used.

プロキシ パスワードの扱い

The Options Connection screen has a new option to 'Prompt for proxy credentials on start up'.
If this option is selected then your proxy password will not be stored on the hard disk.

サイトタブからの再送信

The サイトタブ has a new right click menu item that allows you to resend the request after making any changes to it that you want to.

セッションによるユーザーディレクトリ情報の保持

The last directory used by the user is now stored in the config file so that it is maintained between sessions.
