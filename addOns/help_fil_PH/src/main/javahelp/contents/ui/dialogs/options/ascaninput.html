Opsyon ng aktibong iskan input ng mga vector screen

Ang screen na ito ay pinapayagan ka na i-configure ang aktibong iskan ng input ng mga vector.
Ito ang mga elemento na ang aktibong iskaner ay umaatake.
Ang pag-iskan ng lahat ng nagsusuportang mga elemento ay may katagalan, pero ang hindi pagiskan ng ibang elemento ay magiging dahilan ng mga kahinaan ay maaaring pumalya.

Naiinyek na mga target

Ang mga kahilingan ng mga elemento na ang aktibong iskaner ay ang target:

Ang URL ng Query String & Data Driven ng mga Node

Ang halaga ng mga paris na key sa aplikasyon na URL ng query, ang ie pagkatapos ?.
If Ang data ng mga driven node ay liwanagin sa loob ng isang context ito ay maaaring nasubukan na.
Add URL Query Parameter?
When selected ZAP will add a query parameter to GET requests which did not originally have one. This may increase scan time, however, it may also reveal issues that would otherwise go un-noticed.

POST ng data

Key value pairs in the request POST data.

Plain Body Data

When selected ZAP will process requests with body content and either no or text/plain content-type as a single input/parameter for testing purposes.

Daanan ng URL

Path elements in the request URL, ie the elements separated by /.

Mga header ng HTTP

Request HTTP Headers.
All Requests
Allows to scan the HTTP Headers of all requests. Not just requests that send parameters, through the query or request body.

Cookie ng Data

Request cookies.
Encode Cookie Values
If selected the cookie values will be URL encoded when attacked. By default they will not be encoded - this is new in ZAP 2.15.0, previously they were always encoded.

Build-in Input mga Handler ng Vector

The data formats that the active scanner will target:
    Ang multipart na paraan ng data
    Ang XML tag/attribute
    JSON
    Ang Google Web Toolkit
    Ang OData id/filter

Paganahin ang iskrip input ng mga vector

If this option is selected then the active scanner will use any enabled script input vectors.
Script input vectors are scripts which you have written or imported into ZAP and allow you to target elements which are not supported by default.

Ganoon din ang screen ay pinapahintulutan ka para i-configure ang mga parameter na kung saan ay maaaring baliwalain ng aktibong iskaner.
Refer to Parameter section in Add Alert dialogue for more details about the parameter names.
